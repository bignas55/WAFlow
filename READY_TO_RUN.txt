  Troubleshooting If Needed
═══════════════════════════════════════════════════════════════════════════════

Check if build succeeded:
  docker images | grep waflow

View full build output:
  docker-compose -f docker-compose.1k-users.yml logs

Check app logs:
  docker-compose -f docker-compose.1k-users.yml logs waflow-app-1

Verify Chromium installed:
  docker-compose -f docker-compose.1k-users.yml exec waflow-app-1 chromium-browser --version

View WWJS initialization:
  docker-compose -f docker-compose.1k-users.yml logs waflow-app-1 | grep -i "WWJS\|Browser\|Chromium"
