WHAT WAS BROKEN:
  The TypeScript compiler was running but NOT producing any output files.
  The dist/server directory was created but completely empty.

  Root cause: server/tsconfig.json had incorrect rootDir and include paths
  that prevented TypeScript from finding the source files.

WHAT I FIXED:

1. server/tsconfig.json
   ❌ BEFORE: rootDir: ".."  (tried to compile from parent)
   ✅ AFTER:  rootDir: "."   (compile just the server directory)

   ❌ BEFORE: include: ["../server/**/*", "../shared/**/*", "../drizzle/**/*"]
   ✅ AFTER:  include: ["./**/*.ts"]  (simple, direct pattern)

2. Dockerfile
   ❌ BEFORE: Used pnpm build script (which called tsc with broken config)
   ✅ AFTER:  Direct commands:
              - pnpm exec vite build --config client/vite.config.ts
              - pnpm exec tsc --project server/tsconfig.json
              - Verify both dist/server/index.js and dist/client/index.html exist
