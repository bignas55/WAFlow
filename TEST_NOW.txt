═══════════════════════════════════════════════════════════════════════════════
  QUICK REFERENCE: Common Issues & Fixes
═══════════════════════════════════════════════════════════════════════════════

Issue: "Restarting (0)" for 120+ seconds
Fix: Check logs with: docker-compose logs waflow-app-1 | tail -30

Issue: "ECONNREFUSED" in logs
Fix: Database not ready. Wait 30s, then: docker-compose restart waflow-1

Issue: "Port 3000 already in use"
Fix: Kill the process: kill -9 $(lsof -t -i:3000)

Issue: "Cannot find module"
Fix: Rebuild code:
  npm install
  npm run build:server
  docker-compose restart waflow-1

Issue: Everything keeps failing
Fix: Complete reset:
  docker-compose down -v
  ./setup-docker.sh
  npm run drizzle:generate
  npm run drizzle:migrate
  npm run db:seed
