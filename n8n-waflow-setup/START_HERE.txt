QUICK START (3 steps, 5 minutes)
================================

1. Make installation script executable:
   chmod +x INSTALL.sh

2. Run it:
   ./INSTALL.sh

3. Open in browser:
   http://localhost:5678


WHAT HAPPENS DURING INSTALL
============================

✓ Checks if Docker is installed
✓ Creates .env configuration file
✓ Starts MySQL + n8n containers
✓ Initializes database with all 16 tables
✓ Seeds sample data
✓ Shows you next steps


AFTER INSTALLATION
==================

1. Create admin account at http://localhost:5678 (first login only)

2. Add your Groq API key to .env:
   - Get free key at: https://console.groq.com
   - Edit: nano .env
   - Set: GROQ_API_KEY=gsk_xxxxxxxxxxxx
   - Restart: docker-compose restart n8n

3. Import workflows into n8n:
   - Open http://localhost:5678
   - Click "Projects" → "Import"
   - Select workflow-1-webhook-receiver.json
   - Click "Import"
   - Repeat for workflow-2-template-ai.json

4. Test it works:
   ./test-webhook.sh

5. Read the documentation:
   - README.md (quick overview)
   - SETUP_GUIDE.md (detailed instructions)
   - INDEX.md (file reference)

USEFUL COMMANDS
===============

View logs:
  docker-compose logs -f n8n

Connect to database:
  docker-compose exec mysql mysql -u waflow -pwaflow123 waflow_n8n

Stop everything:
  docker-compose stop

Start everything:
  docker-compose start

Remove everything (keeps data):
  docker-compose down

Show status:
  docker-compose ps
