═══════════════════════════════════════════════════════════════════════════════
  Troubleshooting
═══════════════════════════════════════════════════════════════════════════════

If containers keep restarting:
  Check app logs:
    docker-compose -f docker-compose.1k-users.yml logs waflow-app-1 --tail 50

If database isn't initializing:
  Check MySQL is running:
    docker-compose -f docker-compose.1k-users.yml exec mysql mysqladmin ping

If Chromium isn't working:
  Verify installation:
    docker-compose -f docker-compose.1k-users.yml exec waflow-app-1 chromium-browser --version
